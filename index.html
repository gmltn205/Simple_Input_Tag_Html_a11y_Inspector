웹 접근성 Input 태그 검사 도구

해당 도구는 HTML의 <input> 태그가 웹 접근성 가이드라인(WCAG)을 준수하는지 자동으로 검사하는 도구입니다. 시각 장애인이 사용하는 스크린 리더나 키보드만으로 웹사이트를 이용하는 사용자들이 입력 필드를 올바르게 인식하고 사용할 수 있는지 확인합니다.

접근성 검사 항목

  • Label 연결: 스크린 리더가 입력 필드의 용도를 인식할 수 있는 label 태그 연결
  • ARIA 속성: aria-label,aria-labelledby, aria-describedby 등 보조 기술을 위한 추가 정보
  • 필수 입력 표시: required 속성 사용 시 aria-required의 적절한 설정
  • Title 속성: 마우스 호버 시 도움말을 제공하는 title 속성
    💡 참고로 label, aria-label, aria-labelledby, aria-describedby 속성이 중복 제공되면 화면 리더기는 우선 순위에 따라 선택적으로 설명을 읽습니다.
  • 패턴 안내: pattern 속성 사용 시 placeholder를 통한 입력 형식 안내
  • 검사 타입: search, email, file, text, password 타입만 검사
